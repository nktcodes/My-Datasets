# My-Datasets

 This is a collection of stock market datasets that will be useful for financial analysis.

 Currently the dataset is composed of:

 * AAPL stock data
 * AMZN stock data
 * TSLA stock data
 * NFLX stock data
 * MSFT stock data
 * VIX index data
 * GOOG stock data
 * META stock data
 * AMD stock data
 * HPQ stock data
